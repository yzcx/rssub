1323. Maximum 69 Number #Easy

题目描述

Given a positive integer num consisting only of digits 6 and 9.

Return the maximum number you can get by changing at most one digit (6 becomes 9, and 9 becomes 6).

Example 1:

Input: num = 9669
Output: 9969
Explanation: 
Changing the first digit results in 6669.
Changing the second digit results in 9969.
Changing the third digit results in 9699.
Changing the fourth digit results in 9666. 
The maximum number is 9969.

Example 2:

Input: num = 9996
Output: 9999
Explanation: Changing the last digit 6 to 9 results in the maximum number.

Example 3:

Input: num = 9999
Output: 9999
Explanation: It is better not to apply any change.

Constraints:

  • 1 <= num <= 10^4
  • num’s digits are 6 or 9.

题目解析

从头到尾遍历num的每一位数字，找到第一个6的位置，将其修改为9即可。

C++ 实现

class Solution {
public:
    int maximum69Number (int num) {
        if (num == 9999) return num;
        int factor = 1000;
        while (factor) {
            if (num / factor % 10 == 6) {
                num += 3 * factor;
                break;
            }
            factor /= 10;
        }
        return num;
    }
};

1324. Print Words Vertically #Medium

题目描述

Given a string s. Return all the words vertically in the same order in which they appear in s. Words are returned as a list of strings, complete with spaces when is necessary. (Trailing spaces are not allowed). Each word would be put on only one column and that in one column there will be only one word.

Example 1:

Input: s = "HOW ARE YOU"
Output: ["HAY","ORO","WEU"]
Explanation: Each word is printed vertically. 
"HAY"
"ORO"
"WEU"

Example 2:

Input: s = "TO BE OR NOT TO BE"
Output: ["TBONTB","OEROOE","   T"]
Explanation: Trailing spaces is not allowed. 
"TBONTB"
"OEROOE"
"   T"

Example 3:

Input: s = "CONTEST IS COMING"
Output: ["CIC","OSO","N M","T I","E N","S G","T"]

Constraints:

  • 1 <= s.length <= 200
  • s contains only upper case English letters.
  • It’s guaranteed that there is only one space between 2 words.

题目解析

首先，将字符串s按照空格分割成子串存放到words中，同时记录子串的最大长度max_length；其次，将words中每个子串的每一位存放到res对应位置中，不足max_length的位置用空格补上；最后，将res中每个字符串末尾的空格删去即可。

C++ 实现

class Solution {
public:
    vector<string> printVertically(string s) {
        vector<string> words;
        int max_length = INT_MIN;
        auto beg = s.begin(), end = s.end(), iter = beg;
        while ((iter = find(beg, end, ' ')) != s.end()) {
            words.push_back(s.substr(beg-s.begin(), iter-beg));
            max_length = max(max_length, static_cast<int>(iter-beg));
            beg = iter + 1;
        }
        words.push_back(s.substr(beg-s.begin(), iter-beg));
        max_length = max(max_length, static_cast<int>(iter-beg));
        vector<string> res(max_length);
        for (auto word : words) {
            for (int i = 0; i < word.size(); ++i)
                res[i] += word[i];
            for (int i = word.size(); i < max_length; ++i)
                res[i] += ' ';
        }
        for (auto &str : res) {
            while (str.back() == ' ')
                str.pop_back();
        }
        return res;
    }
};

1325. Delete Leaves With a Given Value #Medium

题目描述

Given a binary tree root and an integer target, delete all the leaf nodes with value target.

Note that once you delete a leaf node with value target, if it’s parent node becomes a leaf node and has the value target, it should also be deleted (you need to continue doing that until you can’t).

Example 1:

Input: root = [1,2,3,2,null,2,4], target = 2
Output: [1,null,3,null,4]
Explanation: Leaf nodes in green with value (target = 2) are removed (Picture in left). 
After removing, new nodes become leaf nodes with value (target = 2) (Picture in center).

Example 2:

Input: root = [1,3,3,3,2], target = 3
Output: [1,3,null,null,2]

Example 3:

Input: root = [1,2,null,2,null,2], target = 2
Output: [1]
Explanation: Leaf nodes in green with value (target = 2) are removed at each step.

Example 4:

Input: root = [1,1,1], target = 1
Output: []

Example 5:

Input: root = [1,2,3], target = 1
Output: [1,2,3]

Constraints:

  • 1 <= target <= 1000
  • Each tree has at most 3000 nodes.
  • Each node’s value is between [1, 1000].

题目解析

使用递归函数bool helper(TreeNode *node, int target)：

  • 如果左节点存在且左节点的递归结果为true，则将左节点赋值为nullptr，右节点同理。

  • 如果左右节点都为nullptr且值为target，说明该节点为需要删除的节点，则返回true。

C++ 实现

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    TreeNode* removeLeafNodes(TreeNode* root, int target) {
        if (helper(root, target)) return nullptr;
        else return root;
    }
    bool helper(TreeNode *node, int target) {
        if (node->left && helper(node->left, target))
            node->left = nullptr;
        if (node->right && helper(node->right, target))
            node->right = nullptr;
        return !node->left && !node->right && node->val == target;
    }
};

1326. Minimum Number of Taps to Open to Water a Garden #Hard

题目描述

There is a one-dimensional garden on the x-axis. The garden starts at the point 0 and ends at the point n. (i.e The length of the garden is n).

There are n + 1 taps located at points [0, 1, ..., n] in the garden.

Given an integer n and an integer array ranges of length n + 1 where ranges[i] (0-indexed) means the i-th tap can water the area [i - ranges[i], i + ranges[i]] if it was open.

Return the minimum number of taps that should be open to water the whole garden, If the garden cannot be watered return -1.

Example 1:

Input: n = 5, ranges = [3,4,1,1,0,0]
Output: 1
Explanation: The tap at point 0 can cover the interval [-3,3]
The tap at point 1 can cover the interval [-3,5]
The tap at point 2 can cover the interval [1,3]
The tap at point 3 can cover the interval [2,4]
The tap at point 4 can cover the interval [4,4]
The tap at point 5 can cover the interval [5,5]
Opening Only the second tap will water the whole garden [0,5]

Example 2:

Input: n = 3, ranges = [0,0,0,0]
Output: -1
Explanation: Even if you activate all the four taps you cannot water the whole garden.

Example 3:

Input: n = 7, ranges = [1,2,1,0,2,1,0,1]
Output: 3

Example 4:

Input: n = 8, ranges = [4,0,0,0,0,0,0,0,4]
Output: 2

Example 5:

Input: n = 8, ranges = [4,0,0,0,4,0,0,0,4]
Output: 1

Constraints:

  • 1 <= n <= 10^4
  • ranges.length == n + 1
  • 0 <= ranges[i] <= 100

题目解析

动态规划。首先，维护数组nranges，存放每个位置的洒水范围；其次，维护数组dp，下标为i的值表示喷洒i至n的范围需要开启的数量，初始值为INT_MAX；然后，从后向前遍历nranges：

  • 如果下标为i的右洒水范围能够到达n，那么dp[i]的值为1。

  • 遍历i+1至n的范围，如果下标为i的右洒水范围能够到达j的左洒水范围而且dp[j]的值不为INT_MAX（说明能够喷洒j至n的范围），那么更新dp[i]为dp[i]和dp[j]+1的较小值。

  • 如果下标为i的左洒水范围能够到达0，那么更新res为res和dp[i]的较小值。

最后，如果res的值未更新过，说明不能喷洒全部范围，返回-1，否则返回res即可。

C++ 实现

class Solution {
public:
    int minTaps(int n, vector<int>& ranges) {
        vector<pair<int, int>> nranges;
        for (int i = 0; i <= n; ++i)
            nranges.push_back({max(0, i-ranges[i]), min(n, i+ranges[i])});
        vector<int> dp(n+1, INT_MAX);
        int res = INT_MAX;
        for (int i = n; i >= 0; --i) {
            if (nranges[i].second >= n) dp[i] = 1;
            for (int j = i + 1; j <= n; ++j) {
                if (nranges[i].second >= nranges[j].first && dp[j] != INT_MAX)
                    dp[i] = min(dp[i], dp[j]+1);
            }
            if (nranges[i].first <= 0)
                res = min(res, dp[i]);
        }
        if (res == INT_MAX) return -1;
        else return res;
    }
};
